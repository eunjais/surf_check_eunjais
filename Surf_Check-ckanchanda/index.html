SURF CHECK

Want to surf, but don't know what the conditions are? Search for a beach below to check the surf forcast!

Diehard lovers of the ocean and surfing, a small group of UC Berkeley data professionals were inspired to create a website that would predict surf forecasts in the Golden State of California. The purpose of this project is to gather maritime weather conditions using an API from worldweatheronline.com to call for the surf forecasts of every and all surfable beaches within California using the given parameters of: latitude, longitude, sunrise, sunset, weather, termperture, weather temperature, cloud cover, wind speed, wave height, and wave size.
